github 提供的是服务器及域名

hexo 是用来写博客（纯静态的博客）没有数据库，没有后台代码！

hexo可以将写博客的工作进行简化，只需要写markdown文件（这个文件中会包含一些信息，比如文章的分类，文章的标签，文章标题，文章的创建时间！），hexo会将markdown文件转换成html文件

步骤：

  1. 需要安装node.js
  2. 通过npm下载hexo
  1. 上面一步完成之后，就有一个命令可以用了 hexo
  2. 创建一个新的文件夹，cd进入这个文件夹
  3. 执行 hexo init，初始化一个博客
  4. 执行 hexo s 在本地可以进行预览
  5. 执行 hexo g 可以将md文件生成为html文件（目录为 当前博客目录下的 public目录）
  6. 想要创建新的博客 hexo new "文章名称"
  7. 要将博客发布到github上， 创建一个静态页面托管的项目
    将public目录下所有的内容全部推送到项目中即可
  8. 以后要写文章，只需要 hexo new hexo g 然后push public目录中的内容即可！
